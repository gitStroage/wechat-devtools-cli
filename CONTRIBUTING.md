# 贡献指南

感谢你对 wechat-devtools-cli 的关注！欢迎通过以下方式贡献：

## 开发环境

```bash
# 克隆仓库
git clone https://github.com/gitStroage/wechat-devtools-cli.git
cd wechat-devtools-cli

# 安装依赖
pnpm install

# 开发模式运行
pnpm dev

# 运行测试
pnpm test

# 构建
pnpm build
```

## 提交规范

使用 [Conventional Commits](https://www.conventionalcommits.org/) 规范：

- `feat:` 新功能
- `fix:` 修复 bug
- `docs:` 文档更新
- `style:` 代码格式（不影响功能）
- `refactor:` 重构
- `test:` 测试相关
- `chore:` 构建/工具相关

## Pull Request 流程

1. Fork 仓库
2. 创建功能分支：`git checkout -b feat/your-feature`
3. 提交更改：`git commit -m 'feat: add your feature'`
4. 推送分支：`git push origin feat/your-feature`
5. 创建 Pull Request

## 报告 Bug

使用 [Issue 模板](https://github.com/gitStroage/wechat-devtools-cli/issues/new?template=bug_report.md) 报告 bug。

## 功能建议

使用 [Feature Request 模板](https://github.com/gitStroage/wechat-devtools-cli/issues/new?template=feature_request.md) 提交功能建议。
